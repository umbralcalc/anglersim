# anglersim

Angling from fish ecosystems: simulations of freshwater angling practices and their effect on UK fish populations.

You can get OpenMPI with instructions [here](https://edu.itp.phys.ethz.ch/hs12/programming_techniques/openmpi.pdf).

In order to set the python path we use direnv: [see here](https://direnv.net/docs/installation.html).
